## 🚀 Featured Projects

This website features my end-to-end analytics projects, highlighting my data pipeline and visualization process:

### 1. EduPlus Customer Churn Analysis
- **Description:** Analyzed online learning platform student data using Power BI to uncover behavioral factors causing service abandonment (churn) and designed targeted retention strategies.
- **Key Deliverable:** Interactive Power BI Dashboard tracking student risk segments.

### 2. Credit Card Customer Churn Dashboard (BankChurners)
- **Description:** Built a full-pipeline analytics dashboard analyzing 10,127 banking customers. Cleaned raw data in Power Query, calculated advanced business metrics using DAX, and built an RFM segmentation model.
- **Key Deliverable:** Comprehensive Power BI Desktop report with concrete customer retention action plans.
